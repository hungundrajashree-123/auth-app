User Authentication System

This project is a simple User Authentication System built using HTML, Tailwind CSS, and JavaScript. It allows users to log in, view their profile, and log out securely using token-based authentication.

---

🚀 Features

- User Login with API integration
- Token stored in localStorage
- Profile page with dynamic user data
- Logout functionality (token removed)
- Redirect after login and logout
- Loading state handling
- Responsive UI using Tailwind CSS
- Live deployment on Netlify

---

 Tech Stack

- HTML5
- Tailwind CSS (CDN)
- JavaScript (Vanilla JS)
- REST API
- GitHub
- Netlify

---

 How It Works

1. User enters login credentials
2. API validates the user
3. Token is saved in localStorage
4. User is redirected to profile page
5. Profile data is fetched using token
6. Logout removes token and redirects to login page
